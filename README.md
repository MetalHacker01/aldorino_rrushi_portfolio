
# Aldorino Rrushi Portfolio

Welcome to the personal portfolio of Aldorino Rrushi, a Marketing Automation Solution Engineer and Web Developer based in Tirana, Albania.

## About This Project

This portfolio showcases my professional experience, certifications, and custom solutions in marketing automation, Salesforce Marketing Cloud, and web development. Built with modern technologies, it serves as a demonstration of my skills, projects, and expertise.

## Features
- **About**: Learn more about my background and professional journey.
- **Experience**: Explore my work history and key roles.
- **Certifications**: View my Salesforce and other industry certifications.
- **Custom Solutions**: Discover unique automation and web solutions I've built.
- **Terminal**: Interactive terminal-style section for fun and engagement.
- **Contact**: Get in touch with me directly.

## Tech Stack
- **Vite**
- **React** (TypeScript)
- **Tailwind CSS**
- **shadcn/ui**

## Getting Started

1. **Install dependencies:**
	```sh
	npm install
	```
2. **Start the development server:**
	```sh
	npm run dev
	```
3. **Build for production:**
	```sh
	npm run build
	```

## Folder Structure
- `src/components/portfolio/` – Main portfolio sections and UI components
- `src/assets/` – Images and static assets (favicon, logo, etc.)
- `public/images/` – Publicly served images and robots.txt
